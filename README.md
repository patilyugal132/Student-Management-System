# 🎓 Student Management System

A Java Web Application developed using Servlet, JSP, JDBC, PostgreSQL, and Apache Tomcat following the MVC Architecture.

---

## 🚀 Features

- User Login & Logout
- Session Management
- Add Student
- View Students
- Update Student
- Delete Student
- Search Student
- Dashboard with Total Student Count
- Bootstrap Responsive UI
- Success Messages
- Delete Confirmation Popup

---

## 🛠️ Technologies Used

- Java
- Servlet
- JSP
- JDBC
- PostgreSQL
- Apache Tomcat
- Bootstrap 5
- Maven
- Git & GitHub

## ⚙️ How to Run

1. Clone the repository

```
git clone https://github.com/patilyugal132/Student-Management-System.git
```

2. Open the project in IntelliJ IDEA

3. Configure PostgreSQL Database

4. Update database credentials in DBConnection.java

5. Run the project using Apache Tomcat
